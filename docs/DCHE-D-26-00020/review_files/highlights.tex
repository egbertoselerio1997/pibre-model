\section*{Highlights}

\begin{itemize}
    \item ICSOR enforces exact mass conservation and non-negativity in one surrogate.
    \item Only model with zero conservation and non-negativity violations on all test samples.
    \item Lowest mean held-out RMSE at the smallest retained dataset size among all ten models.
    \item Lowest normalized area under the RMSE learning curve across dataset sizes.
    \item Smallest train-validation RMSE gap among competitively accurate surrogate models.
\end{itemize}
