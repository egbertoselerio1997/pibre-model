# ASM2D-TSN Workbook Reconstruction Specification

This document describes the current canonical workbook at `data/asm2d-tsn/asm2d_tsn_workbook.xlsx`. It reflects the current ASM2D-TSN implementation aligned to schema version `2.0`, uses the `S_O` state name, and exposes only the five measured composites `COD`, `TN`, `TKN`, `TP`, and `TSS`.

The workbook itself is formula-driven and was generated by the repository code. Because OpenPyXL does not calculate Excel formulas, the evaluated matrix values listed below come from the current numeric expansion returned by `get_asm2d_tsn_matrices()`, not from cached Excel workbook values.

## 1. Scope and reconstruction rules

- Recreate one workbook named `asm2d_tsn_workbook.xlsx`.
- Keep exactly three worksheets in this order: `stoichiometric_matrix`, `composition_matrix`, `parameter_table`.
- Do not add extra worksheets, merged cells, hidden rows, or hidden columns.
- Blank matrix cells are intentionally blank. Do not replace blanks with `0`, `0.0`, or `=0`.
- The `parameter_table` must be created before the other sheets because the matrix sheets reference absolute cells in column `E`.
- Reproduce formulas exactly as shown, including absolute references such as `parameter_table!$E$2`.
